NHL Analytics

A set of connected views for reading NHL hockey at different levels of depth — from tonight's schedule and live game tracking to player-level stats, line combinations, expected goals models, and team projections. Every view uses the same team and season filters, so pivoting from game to player to line is one click.

Game Discovery
Schedule
Full calendar view for any team, any season back to 1917. Click any game to open the full game report.
Live Games
Fast-scan view of games in progress — score, period, and a direct link into each game's live data.
Standings
Points, GF/GA, differentials and win streaks for all teams. Filter by season and game state.
Game Projections
Model-based pre-game outlooks using roster context, recent form, and goaltending assignments.
Player & Team Analysis
Skaters
Player cards, shot maps, trend charts, RAPM, and projections. Exportable as PNG for any view.
Goalies
GSAA, GSAx, save percentage trends, and model-informed workload context.
Teams
Team-level dashboards across play-driving, shooting, and projection metrics. Roster-stacked charts.
Line Tool
Pick any player combination. View zone heat maps, WOWY breakdowns, and line usage tables.
